## ✨ Key Features
* **Modular Learning:** Har class ek alag JS file mein hai (`class_01.js`,`class_02.js`, etc.).
* **Interactive UI:** GSAP animations aur Lottie files ka smooth use.
* **Data Visualization:** Chart.js se aap apni progress dekh sakte hain.
* **Storage:** Aapka score `localStorage` mein save rehta hai.
